# How to mod your kindle
> **Winterbreak** — The most common jailbreak (probably)
### Installing Winterbreak

> [!WARNING]
> Please enable Airplane mode before and after jailbreaking!

I've made a .sh to automatically install WinterBreak and KUAL, To use it, [download the sh here.](https://github.com/Pixel8ash/WinterBreak-Auto/raw/refs/heads/main/winterbreak.sh) once done, open it using ./WinterBreak-Auto.sh, once prompted to, please plug in your kindle. You will be asked if you want to proceed. Select Y for yes and N for no. Once it says 'All done! Check the guide for further instructions!' open the Kindle store. It will say to disable airplane mode, please disable it. Tap the WinterBreak
icon and wait. Once done it should say 'Now you are ready to install the hotfix.', please enable airplane mode again. Rerun the .sh once more. This will install the hotfix. Please go to Settings > **⋮** and Update your kindle. Now we can go to KUAL! Rerun once more and then wait. Open KUAL and select Rename OTA Bins and Rename. You are now jailbroken and now can turn off airplane mode!
